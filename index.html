闭包
了解this的机制，闭包的引用。闭包的使用能在当前作用域引用外部作用域中的变量。因为 函数 是 JavaScript 中唯一拥有自身作用域的结构，因此闭包的创建依赖于函数。
        function bar(num){
        var qunt = num;
        return{
            init:function(){
            qunt++;
          },
            get:function(){
            return qunt;
          }
      }
      }
    var foo = bar(2);
    foo.qunt();
    foo.get();
其中： 
        foo.get() //5
